Essential Tools for Crypto Privacy

Curated resources with significant activity. Updated: Feb 2026.

  • Sparrow Wallet

    Desktop wallet with coin control, labels, Whirlpool/Wasabi support, and BIP47/PayJoin flows.

    BitcoinWalletCoin control
  • Wasabi Wallet

    WabiSabi CoinJoin + cross-platform GUI.

    BitcoinCoinJoin
  • JoinMarket

    Incentive-based CoinJoin marketplace with maker/taker liquidity.

    BitcoinCoinJoinP2P
  • Samourai Dojo

    Self-hosted Bitcoin backend for powering Samourai features (Cahoots, Whirlpool) via Tor.

    BitcoinBackend
  • Whirlpool CLI

    Command-line Whirlpool client so you can mix UTXOs on your own node with deterministic fees.

    BitcoinCoinJoinCLI
  • Chaincase

    iOS wallet focused on Whirlpool mixing, PayNym contacts, PayJoin, and Tor networking.

    BitcoinWalletMobile
  • Feather Wallet

    Modern Monero desktop wallet with Tor/I2P transport, coin control, and hardware wallet support.

    MoneroWallet
  • Monero CLI

    Reference Monero node and wallet CLI with ring signatures, view-only modes, and RPC APIs.

    MoneroNode
  • Monero GUI

    Official Qt-based GUI wallet bundling hardware support, node management, and Ledger/Trezor flows.

    MoneroWalletDesktop
  • Monerujo

    Android Monero wallet (Monerujo) with coin control, Tor/I2P networking, and QR/PayCode tools.

    MoneroWalletMobile
  • Cake Wallet

    Open-source iOS/Android wallet for Monero & BTC with Tor routing and in-app swap bridges.

    MoneroWalletMobile
  • EigenWallet

    Cross-platform wallet with built-in Tor routing and peer-to-peer BTC ↔ XMR atomic swaps.

    MoneroAtomic swap
  • Gliese.io

    Non-custodial UTXO analysis + transaction simulator built on Bitcoin/Litecoin full nodes.

    BitcoinUTXOAnalysis
  • Wagyu API

    Public API for XMR swaps/bridging (XMR ↔ XMR1/USDC) with optional integrator fees and analytics.

    MoneroAPIBridge
  • THORChain

    Non-custodial cross-chain DEX for native BTC and other L1 assets, using node-bonded liquidity pools instead of wrapped tokens.

    Cross-chainDEX
  • Crypto Privacy Scorecard

    Community-maintained scorecard rating wallets and services on KYC, surveillance, and privacy defaults.

    ResearchScorecard
  • Privacy Watch

    Catalog of Bitcoin privacy best practices and service audits highlighting deanonymization risks.

    ResearchGuides
  • Haveno DEX

    Non-custodial Tor-based DEX for fiat ↔ Monero trades; open-source Java stack with multisig escrows.

    MoneroP2P
  • RetoSwap

    Desktop client built on Haveno offering Tor-native Monero swaps with fiat/crypto payment rails.

    MoneroP2P
  • BasicSwap DEX

    Particl-powered atomic swap DEX for BTC, XMR, FIRO, and other privacy coins over Tor/I2P.

    Cross-chainAtomic swap
  • Bisq

    Tor-based desktop DEX for fiat ↔ BTC trades secured by bonded multisig escrows.

    BitcoinP2P
  • RoboSats

    Tor-native Lightning P2P exchange with robot avatars, escrowed orders, and no account persistence.

    BitcoinP2PTor
  • Lnp2pBot

    Telegram-native escrow bot coordinating BTC trades via Lightning invoices or on-chain holds.

    BitcoinP2PBot
